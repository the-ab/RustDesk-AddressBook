0.5.32-github-release-update-default
[de]
- GitHub-Releases des Projekts als feste Standardquelle für signierte Online-Updates eingerichtet.
- latest.txt aus dem neuesten veröffentlichten Release bestimmt die Update-ZIP; SHA-256-Manifest und Ed25519-Signatur werden aus demselben Release geladen.
- Eigene Updatequellen bleiben unterstützt; RAB_UPDATE_BASE_URL=disabled schaltet Online-Prüfungen ausdrücklich ab.
[en]
- Configured the project GitHub Releases endpoint as the default signed online-update source.
- latest.txt from the latest published release selects the update ZIP; the SHA-256 manifest and Ed25519 signature are downloaded from the same release.
- Custom update sources remain supported; RAB_UPDATE_BASE_URL=disabled explicitly disables online checks.
