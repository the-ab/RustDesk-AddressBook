RustDesk SSH-Import

Holt per SSH eine konsistente Snapshot-Kopie der RustDesk-Serverdatenbank. Empfohlen ist ein dedizierter SSH-Key mit Forced Command auf dem RustDesk-Server.

{% set ssh = get_ssh_import_settings() %}
Der Key muss aus Sicherheitsgründen unter /data liegen, empfohlen: /data/ssh/rustdesk_import_ed25519.
Bei einem eingeschränkten Key mit command="/usr/local/sbin/rab-rustdesk-db-export" leer lassen.
