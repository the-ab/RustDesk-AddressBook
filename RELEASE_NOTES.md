# RustDesk AddressBook 0.5.29 – Standardized documentation languages

## Changes

- All regular Markdown files (`*.md`) now use English as the default language.
- Added matching German editions with the `*.de.md` suffix for every documentation file.
- `README.md` and `RELEASE_NOTES.md` are English; `README.de.md` and `RELEASE_NOTES.de.md` contain the German editions.
- The admin guide, security report, third-party notices, and updates-directory instructions follow the same convention.
- Updated internal documentation references and version examples to version 0.5.29.

## Language convention

| English (default) | German |
|---|---|
| `README.md` | `README.de.md` |
| `ADMIN-GUIDE.md` | `ADMIN-GUIDE.de.md` |
| `RELEASE_NOTES.md` | `RELEASE_NOTES.de.md` |
| `SECURITY-REPORT.md` | `SECURITY-REPORT.de.md` |
| `THIRD-PARTY-NOTICES.md` | `THIRD-PARTY-NOTICES.de.md` |
| `updates/README.md` | `updates/README.de.md` |

This release changes documentation, file naming, and version references only. Application features and security controls remain unchanged.
